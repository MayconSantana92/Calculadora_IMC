## 📋 **Funcionalidades**

- 🔹 **Cálculo do IMC**: Informe seu peso (em kg) e altura (em metros) para obter o índice.
- 🔹 **Classificação do IMC**:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade
- 🔹 **Design Responsivo**: Funciona perfeitamente em diferentes tamanhos de tela.
- 🔹 **Informações Extras**: Link para saber mais sobre o cálculo do IMC.
